JQuery
--------
How jquery is related/ different from Javascript

JQuery is an application of javascript.

It simplifies the way we code javascript programs.

Ex:
	in javascirpt:
		document.getElementById("div1").innerHTML="<b>This is output</b>";

	in jquery:
		$("#div1").html("<b>This is output</b>");

Look at the simplicity of jquery

javascript				jquery
value				val()
value="India"			val("India")
innerText				text
innerHTML			html
.style.color=red			.css("color","red")




function loadDoc() {
  var xhttp = new XMLHttpRequest();
  xhttp.onreadystatechange = function() {
    if (this.readyState == 4 && this.status == 200) {
      document.getElementById("demo").innerHTML =
      this.responseText;
    }
  };
  xhttp.open("GET", "ajax_info.txt", true);
  xhttp.send();
}


$(document).ready(function(){
    $("button").click(function(){
        $.ajax({url: "demo_test.txt", success: function(result){
            $("#div1").html(result);
        }});
    });
});

	selector.event(handler)

	selector:		p		(element selector)

	handler:		function(){}

this is anonymous function that means, there is no name for the function
