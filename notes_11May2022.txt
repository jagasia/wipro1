Spring security + JWT
----------------------------
1) spring boot app
2) dependencies
	spring web
	spring security
	jjwt
	spring data jpa
	mysql driver
3) we create a rest api
	@RestController

	we will create
		entities
		repositories
		service classes
		rest controllers
4) we will configure this spring boot app with 
	jwt filter
		will intercept every request comes, and look for a header "Authorization".

	in spring configuration		WebMvcConfigurerAdapter
		we will permitAll for certain URLs So that the user will be able to login and get JWT token
		logging in means, an URL for login must be allowed

5) we will define
	JwtUtil class
		that have methods to
			generateToken
			createToken
			validateToken
			extractUserName

6) after this,
	we will create an Angular project for frontend

learn how to use this jwt token whenever we access the rest api
